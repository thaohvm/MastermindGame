Game Rules

This is a game where you try to guess the number combinations.

  • Starting by select the number of digits that you want to guess and number of attempts you wish.
  • You must enter full combination that you guess.
  • At each attempts, you will be provided feedback whether you had guess a number correctly, and/or a number and digit correctly.
  • You must guess the right number combinations within the number of requested attempts to win the game.
  • Duplicate numbers are allowed in the combination.
